# Comprehensive Compiler Demo

This demo file exercises **all** features of the ByteCode Compiler.

## Running the Demo

```bash
./build/compiler examples/comprehensive_demo.txt
```

## Features Demonstrated

### 1. **Variables & Arithmetic**
- Variable declarations (`let`)
- Addition, subtraction, multiplication, division, modulo

### 2. **String Literals**
- String storage and printing

### 3. **Conditional Statements**
- `if` statements
- Nested conditions

### 4. **While Loops**
- Loop control with conditions

### 5. **For Loops**
- Initialization, condition, increment
- Loop counters

### 6. **Break Statement**
- Early loop termination

### 7. **Continue Statement**
- Skip to next iteration

### 8. **Function Definitions**
- Parameter passing
- Return values
- Multiple functions

### 9. **Recursive Functions**
- Factorial calculation
- Stack frame management

### 10. **Array Literals**
- Array initialization

### 11. **Array Indexing**
- Element access

### 12. **Array Assignment**
- Modifying array elements

### 13. **Nested Arrays**
- Multi-dimensional arrays
- Nested indexing

### 14. **Array Operations**
- Looping through arrays
- Sum calculation

### 15. **Comparison Operators**
- ==, !=, <, <=, >, >=

### 16. **Complex Algorithm**
- Bubble Sort implementation

### 17. **Functions with Arrays**
- Array parameters
- Finding maximum value

### 18. **Nested Loops**
- Multi-level iteration

## Expected Output

The program prints:
- Test results for each feature
- Properly formatted arrays `[1, 2, 3]`
- Computed values
- Sorted array results

All features should execute successfully!
